## Quick Start

### Step 1. Clone the Repo

```shell
$ git clone https://github.com/Zophyr/Lacomp
```

### Step 2. Install dependence

```shell
$ cd Lacomp/
$ npm install
```

### Step 3. Link

```shell
$ react-native link
```

### Step 4. RUN!

```shell
$ react-native run-ios
```
